# Recruitment-Client-Rest-API-Pets
Prosta aplikacja backendowa w Laravelu integrująca się z zewnętrznym REST API do zarządzania zasobem „pet”. Projekt umożliwia dodawanie, edycję, pobieranie i usuwanie danych oraz zawiera podstawowy interfejs formularzy i obsługę błędów API, walidacji oraz problemów z połączeniem.
